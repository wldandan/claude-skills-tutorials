## 科学级的因果分析

**超越相关性，回答真正的"为什么"**

```
❌ 相关性: "CPU 升高时，响应变慢"
   → 可能是巧合，可能是反向因果

✅ 因果推断: "数据库锁等待导致 CPU 升高，
            进而导致响应变慢"
   → 真正的根因，可干预的方向
```

**核心能力**
- 🧪 **反事实推理**："如果没升级数据库，故障还会发生吗？"
- 🔄 **干预模拟**：在安全环境中预测变更影响
- 📊 **A/B 测试框架**：验证优化效果的统计显著性
